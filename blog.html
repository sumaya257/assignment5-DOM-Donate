Important FAQ

What is the DOM?

The DOM (Document Object Model) is a programming interface for HTML and XML documents that represents the page's structure as objects and nodes.

How do you select an element by its ID in the DOM?

Use document.getElementById('elementID') to select an element by its ID.

How do you add a new element to the DOM?

You can use document.createElement('tagName') and then append it using parentElement.appendChild(newElement).

How can you remove an element from the DOM?

Use element.remove() to remove an element from the DOM.
